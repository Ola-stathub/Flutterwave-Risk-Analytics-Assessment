# Fraud Detection Assessment

## Overview

This repository contains a fraud detection assessment completed using a synthetic payment transactions dataset. The project covers SQL-based feature engineering, exploratory data analysis (EDA), feature engineering, machine learning model development, and business recommendations for fraud risk management.


## Project Workflow

1. Data Cleaning and Preparation
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Data Preprocessing
5. Model Training and Evaluation
6. Business Recommendation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* PostgreSQL (SQL Feature Engineering)

## Model Performance

* Accuracy: 99%
* Fraud Precision: 100%
* Fraud Recall: 68%
* Fraud F1-Score: 81%
* ROC-AUC: 0.884
